# AI Resume Screening Platform

An AI-powered Resume Screening and Candidate Ranking Platform built using Python, Streamlit, NLP, and Machine Learning.

## Features

- ATS Resume Screening
- Resume Ranking System
- Candidate Skill Matching
- Domain Prediction
- AI Recommendations
- Recruiter Dashboard
- Interview Question Generator
- Analytics & Visualizations

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- Plotly
- NLP

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
